# Release Notes - v1.0.0

## Summary

Initial public release of the IoT Device Digital Twin Dashboard.

## Included in this release

- Flask backend for simulated device telemetry
- Browser-based dashboard interface
- Real-time polling workflow
- Temperature and humidity chart visualization
- Device status indicator and alert logic
- Professional README and repository structure

## Intended audience

This release is intended for developers, researchers, and engineers interested in IoT monitoring, digital twin concepts, and dashboard-oriented software projects.

## Next planned improvements

- WebSocket support
- Multi-device architecture
- Database-backed telemetry history
- MQTT integration
